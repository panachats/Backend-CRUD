ตารางพนักงาน

ตารางนี้เป็นการเก็บข้อมูลของพนักงาน ได้แก่

  1. EmpID : เก็บ ID ของพนักงาน เช่น 1 เก็บเป็นตัวเลข
  2. EmpName : เก็บ ชื่อจริง ของพนักงาน เช่น ปณชัช เก็บเป็นข้อความ
  3. EmpLastname : เก็บ นามสกุลของพนักงาน ของพนักงาน เช่น เอี่ยมน้ำ เก็บเป็นข้อความ
  4. EmpAge : เก็บ อายุ ของพนักงาน เช่น 20 เก็บเป็นตัวเลข
  5. EmpAdress : เก็บ ที่อยู่ ของพนักงาน เช่น 138/2 อ. ท่าซัก เก็บเป็นข้อความ
  6. EmpSalary : เก็บ เงินเดือน ของพนักงาน เช่น 30000 เก็บเป็นตัวเลข
  7. EmpPosition : เก็บ ตำแหน่ง ของพนักงาน เช่น โปรแกรมเมอร์ เก็บเป็นข้อความ
  8. Activeflag : เก็บ สถานะการใช้งานของผู้เข้าใช้ระบบ 1 หมายความว่า ใช้งาน แต่ถ้าเป็นเลขอื่นจะขึ้นว่า ไม่ใช้งานเก็บเป็นข้อความ
